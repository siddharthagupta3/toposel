Place your new campaign images in this folder using these exact names:

1) hero-sleek-chic.png
- Use the first poster image (white background, standing model)
- Used in hero section

2) product-nuova-yellow.png
- Use the yellow NUOVA poster image
- Used as first product card image

3) product-pastel-pink.png
- Use the pastel pink collage image
- Used as second product card image

Notes:
- PNG/JPG both are okay, but if you use JPG then update path in settings and defaults.
- Recommended minimum width: 900px for each image.
